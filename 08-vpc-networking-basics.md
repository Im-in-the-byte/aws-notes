# 08 VPC Networking Basics

## What is VPC?
- **VPC (Virture Private Cloud)**: A logically isolated section of the AWS cloud where you can launch AWS resources in **vertical network** that you define.
- You have full control over:
  - IP address ranges
  - Subnets
  - Route tables
  - Network gateways
  - Security settings
